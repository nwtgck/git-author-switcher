# Author switcher for git

## Example usage

```bash
$ ./switch-author nwtgck
```

or

```bash
$ ./switch-author nw
```

(Smart search can solve by first letters `nw`)

## Author information

Author information is written in `authors.yaml`.

```yaml
nwtgck:
  name : Ryo Ota
  email: nwtgck@gmail.com
```
